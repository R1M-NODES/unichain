<b> Install Unichain </b>

```
bash <(curl -s https://raw.githubusercontent.com/R1M-NODES/unichain/master/install.sh)
```
Cheat Sheet

```
docker logs unichain-node-execution-client-1

docker-compose down
docker-compose up -d
```
Delete Node

```
cd unichain-node
docker-compose down
sudo rm -r unichain-node
```
